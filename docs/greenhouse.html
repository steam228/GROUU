GrouuPro — the V0 greenhouse

Circa 2013–2014, GrouuPro was a fully-built, full-stack IoT greenhouse — a physical structure, a custom control board, an Arduino front-end and a real-time web dashboard. It is the direct ancestor of everything GROUU became: the same sense → serve → store → visualise architecture the current stack rebuilds with modern tools. Sole-authored, with build assistance from Rui Lopes.

How it worked

The architecture GROUU still uses

An Arduino streamed analog sensor values over serial to a Node.js server on a Raspberry Pi, which stored them and pushed live updates to an Angular dashboard. Ten years on, the current stack is the same shape — only the parts changed.

Hardware

A real greenhouse, not a breadboard

GrouuPro was engineered and built as a product — glazed structure, plumbing, and a custom control board — with the design work in SolidWorks, Rhino, SketchUp and AutoCAD.
