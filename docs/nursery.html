Goals & context

Why a nursery

Propagation is a bottleneck for a small grower: seedlings need stable warmth, humidity and watering before they go out. The nursery is the V3 node that manages that stage — and a testbed for climate control on the shared stack.

Where it sits: a V3 build on the same sense → node → stack path as WellBouy, descending from the archived V0 greenhouse but reduced to the propagation stage and the current node/stack architecture.

Reuses the basic node

No firmware is written from scratch: the nursery node follows the completed basic sensor node (or its LoRaWAN peer), publishing JSON/CayenneLPP to the same broker. Node-specific config lands here.
