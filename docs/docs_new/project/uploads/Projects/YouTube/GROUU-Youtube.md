# 🌽 GROUU YouTube Channel

← [[../MOC-Projects|🛠️ Projects]] · [[../../HOME|🌿 HOME]]

---

## Concept

A YouTube series documenting the GROUU project — applying 15+ years of open-source agriculture research to a real farm. Honest, personal, no-filter.

> *"Uma série que documenta honestamente como se transforma conhecimento teórico em acção prática."*

---

## Format

| Element | Description |
|---|---|
| Duration | 10–15 min per episode |
| Style | Blend of narrative + documentary (Casey Neistat influenced) |
| Filming | Main camera (tripod/studio) + iPhone (field/close-ups) |
| Language | Portuguese |
| Tone | Brutally honest — show failures, not just wins |

---

## Channel Goals

1. **Compromisso** — commit publicly to doing this properly
2. **Rendimento** — eventual income stream from channel
3. **Partilha de Conhecimento** — share what works (and doesn't) with the open-source farming community

---

## Episode Structure (recurring)

```
[OPENING SEQUENCE] — Rustic music, walking the land
     ↓
[MAKER DESK] — Direct to camera: context + reflection
     ↓
[FIELD SEGMENT] — iPhone POV, close-ups, honest state of things
     ↓
[MAKER DESK] — Analysis, what's next
     ↓
[CLOSING] — Preview of next episode + CTA
```
